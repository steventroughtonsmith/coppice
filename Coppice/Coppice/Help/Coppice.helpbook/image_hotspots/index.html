Image Hotspot

Starting with Coppice 2022.1 you can add hotspots to images. Hotspots allow you to create links from an Image Page to another page or external link.

Creating a Rectangle Hotspot
  1. Start editing the Image Page you wish to create a hotspot on
  2. Select the Hotspot Mode icon {[image|image_hotspots_mode_icon.jpg|style="width:37px"]} from the image inspector
  3. Select the Rectangle hotspot option
  4. Click and drag over the area you wish to create the hotspot
Creating an Oval Hotspot (Pro only)
  1. Start editing the Image Page you wish to create a hotspot on
  2. Select the Hotspot Mode icon {[image|image_hotspots_mode_icon.jpg|style="width:37px"]} from the image inspector
  3. Select the Oval hotspot option
  4. Click and drag over the area you wish to create the hotspot
Creating a Polygon Hotspot (Pro only)
  1. Start editing the Image Page you wish to create a hotspot on
  2. Select the Hotspot Mode icon {[image|image_hotspots_mode_icon.jpg|style="width:37px"]} from the image inspector
  3. Select the Polygon hotspot option
  4. Click where you want the first point of the polygon to go
  5. Move the mouse cursor to where you want the next point and click again.
  6. Continue until you have the shape you want. Then click back on the first point to finish
    Alternatively, double click the final point to finish editing
Editing a Hotspot
  1. Start editing the Image Page you wish to create a hotspot on
  2. Select the Hotspot Mode icon {[image|image_hotspots_mode_icon.jpg|style="width:37px"]} from the image inspector
  3. Click and drag on a hotspot handle to change the hotspot's shape
  4. Click and drag on the centre of the hotspot to move it
Removing a Hotspot
  1. Start editing the Image Page you wish to create a hotspot on
  2. Select the Hotspot Mode icon {[image|image_hotspots_mode_icon.jpg|style="width:37px"]} from the image inspector
  3. Click on the hotspot you wish to remove
  4. Press the delete key to remove the hotspot
You can also shift-click to select multiple hotspots before removing
