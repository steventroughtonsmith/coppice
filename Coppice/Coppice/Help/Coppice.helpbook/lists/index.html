Lists

Text Pages in Coppice have support for creating lists. Lists can either be bulleted or numbered and can be nested to create hierarchies of items.

Create a new list
  1. Start editing the Text Page
  2. Click on the List popup in the Paragraph inspector
  3. Choose a list type from the options
Convert existing text to a list
  1. Start editing the Text Page
  2. Select the paragraphs to convert into list items
  3. Click on the List popup in the Paragraph inspector
  4. Choose a list type from the options
Removing a list
  1. Start editing the Text Page
  2. Put the text cursor in the list you wish to remove
  3. Click on the List pop up in the Paragraph inspector
  4. Choose None from the list
Add a new item to a list
  1. Start editing the Text Page
  2. Put the text cursor at the end of the list item you wish to add an item after
  3. Hit the Return key on your keyboard to add a new item
Nest a list item
  1. Start editing the Text Page
  2. Put the text cursor in the list item you wish to nest
  3. Hit the Tab key
You can select multiple items to nest them all at once
Unnest a list item
  1. Start editing the Text Page
  2. Put the text cursor in the list item you wish to unnest
  3. Hit the Shift-Tab
You can select multiple items to unnest them all at once
Change the list type
  1. Start editing the Text Page
  2. Put the text cursor in the list you wish to change
  3. Click on the List pop up in the Paragraph inspector
  4. Choose the new list type from the options
Customise the list type
  1. Start editing the Text Page
  2. Put the text cursor in the list you wish to customise
  3. Click on the List pop up in the Paragraph inspector
  4. Choose Custom… from the list
  5. In the sheet that appears, choose a Bullet/Number
  6. Optionally, set a Prefix and/or Suffix for each item
  7. Click OK
Change the starting number of a list
  1. Start editing the Text Page
  2. Put the text cursor in the numbered list you wish to customise
  3. Click on the List pop up in the Paragraph inspector
  4. Choose Custom… from the list
  5. Click on the Starting number stepper to increase or decrease the starting number
  6. Click OK
