Cropping an Image

Since version 2022.1 it has been possible to crop an image in Coppice. Cropping in Coppice is completely non-destructive, so you can always go back and change your mind later.

Cropping an image
  1. Start editing an image page
  2. Select the Crop Mode icon {[image|cropping_an_image_mode_icon.jpg|style="width:37px"]} from the image inspector
  3. Drag the resize handles around the image to crop the image
  4. Click and drag in the middle of the crop region to move the crop area around
  5. Switch out of Crop Mode to view the cropped image
If you crop an image that already contains hotspots, any hotspots not entirely in the cropped region will not be displayed
