Set the Image of an Image Page

Image Pages can contain a single image. This image can be a PNG, JPEG, non-animated GIF, BMP, or TIFF.

Change the image
  1. Start editing the Image Page
  2. Do one of the following:
    • Image Files: Drag an image file onto the editor
    • Image Content: Drag an image from another app onto the editor
