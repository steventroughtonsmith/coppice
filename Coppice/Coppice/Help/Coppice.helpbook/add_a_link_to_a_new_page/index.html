Add a Link to a New Page

Coppice lets you create a new page as part of adding a link. How you create links differs by page type

Text Page
  1. Start editing the Image Page you wish to create a link in
  2. Select the text on that Page you wish to turn into a link
  3. Click the control in the Link inspector
    Alternatively select Page > Create Link… (⌘L) from the menu bar
  4. Start typing the the title you want for the new page
  5. Click on the Page type you want to create from the Create New… section of the auto-complete
    Alternatively use the up and down arrow keys to choose a Page hit the enter key select a Page type
The new Page will be created in the same folder as the Page you are currently editing
You can also use Page > Create New Linked Page and select a Page type to create a new page titled with the selected text
Image Page
  1. Start editing the Image Page you wish to create a link in
  2. Select or {[link|image_hotspots|create a hotspot]} on the image
  3. Click the control in the Link inspector
    Alternatively select Page > Create Link… (⌘L) from the menu bar
  4. Start typing the the title you want for the new page
  5. Click on the Page type you want to create from the Create New… section of the auto-complete
    Alternatively use the up and down arrow keys to choose a Page hit the enter key select a Page type
The new Page will be created in the same folder as the Page you are currently editing
