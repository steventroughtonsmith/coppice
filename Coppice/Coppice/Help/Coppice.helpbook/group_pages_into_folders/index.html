Group Pages Into Folders

If you have a Coppice Pro subscription, Pages in the sidebar can be grouped into Folders. This can help you better organise the Pages in your document

Create a Folder
  1. Select File > New Folder from the menu bar
  2. Type a name for the Folder
  3. Hit the enter key
Create a Folder from a selection
  1. Select the Pages and Folders in the sidebar you want to group into a new Folder
  2. Click the + button at the bottom of the sidebar
  3. Select Create Folder From Selection
  4. Type a name for the Folder
  5. Hit the enter key
Add Pages to a Folder
  1. Find the Page(s) you want to add to the Folder in the sidebar and select them
  2. Click and drag the Pages. An indicator should appear as you drag them
  3. Move the mouse cursor over the folder. Release the mouse button to add the pages to the folder
    Alternatively, move the mouse cursor to a location in the Folder to insert the pages at that point
Remove Pages from a Folder
  1. Find the Page(s) you want to add to the Folder in the sidebar and select them
  2. Click and drag the Pages. An indicator should appear as you drag them
  3. Release the mouse button when the indicator is where you want the Pages to be
Rename a folder
  1. Double click on the Folder you want to rename in the sidebar
  2. Type in a new name
  3. Hit the enter key
