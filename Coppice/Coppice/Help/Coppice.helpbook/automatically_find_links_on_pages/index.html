Automatically Find Links on Pages

If you have a {[link|purchasing_coppice_pro|Coppice Pro subscription]}, Coppice can help find relationships between Pages for you and automatically create links

Enable auto-linking
  1. Select Coppice > Preferences… (⌘,) from the menu bar
  2. Click on the General tab
  3. Check Auto-detect page titles
Exclude a Page from auto-linking
  1. Select the Page you want to exclude, either from the sidebar or in a Canvas
  2. Uncheck the Allows Auto-Linking checkbox in the Page Info inspector
