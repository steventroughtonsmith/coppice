Create a Link

Links allow you to define relationships between Pages. Currently Coppice allows you to take any text in a page and use it to create a link to any other page.

Create a link to an existing Page
  1. Start editing the text Page you wish to create a link in
  2. Select the text on that Page you wish to turn into a link
  3. Click the control in the Link inspector
    Alternatively select Page > Create Link… (⌘L) from the menu bar
  4. Start typing to search for the Page you want
  5. Click on the Page you want to link to
    Alternatively use the up and down arrow keys to choose a Page hit the enter key to create a link to it
Create a link to a new Page
  1. Start editing the text Page you wish to create a link in
  2. Select the text on that Page you wish to turn into a link
  3. Click the control in the Link inspector
    Alternatively select Page > Create Link… (⌘L) from the menu bar
  4. Start typing the the title you want for the new page
  5. Click on the Page type you want to create from the Create New… section of the auto-complete
    Alternatively use the up and down arrow keys to choose a Page hit the enter key select a Page type
The new Page will be created in the same folder as the Page you are currently editing
You can also use Page > Create New Linked Page and select a Page type to create a new page titled with the selected text
Edit a link
  1. Start editing the text Page you wish to edit a link in
  2. Move the text cursor to the link you want to edit
    3. You can option-click on a link to move the cursor there without activating the link
  3. Click the control in the Link inspector
    Alternatively select Page > Create Link… (⌘L) from the menu bar
  4. Start typing to search for the Page you now want to link to
  5. Click on the Page you want to link to
    Alternatively use the up and down arrow keys to choose a Page hit the enter key to create a link to it
Remove a link from a Page
  1. Start editing the text Page you wish to edit a link in
  2. Move the text cursor to the link you want to edit
    3. You can option-click on a link to move the cursor there without activating the link
  3. Click the {[image|create_a_link_remove_button.jpg|style="width:31px;" alt="Button containing a cross"]} button on the right of the Link inspector
