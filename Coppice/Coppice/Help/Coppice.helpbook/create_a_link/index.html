Create a Link

Links allow you to define relationships between Pages. Currently Coppice allows you to take any text in a page and use it to create a link to any other page.

Create a link to an existing Page
  1. Start editing the text Page you wish to create a link in
  2. Select the text on that Page you wish to turn into a link
  3. Click the Link to Page {[image|create_a_link_toolbar_icon.jpg|style="width:20px"]} toolbar item
    Alternatively select Page > Link to Page (âŒ˜L) from the menu bar
  4. The Page selector will appear. Start typing to search for the Page you want
  5. Click on the Page you want to link to
    Alternatively use the up and down arrow keys to choose a Page hit the enter key to create a link to it
Create a link to a new Page
  1. Start editing the text Page you wish to create a link in
  2. Select the text on that Page you wish to turn into a link
  3. Select Page > Create New Linked Page from the menu bar
    Alternatively right click and select from the Context Menu
  4. Select the type of Page to create
The new Page will be created in the same folder as the Page you are currently editing
Remove a link from a Page
  1. Start editing the text Page you wish to create a link in
  2. Right click on the link you want to remove
  3. Select Remove Link from the context menu
