Create a Link

Links allow you to define relationships between Pages. How you create links differs by page type

Text Page
  1. Start editing the Text Page you wish to create a link in
  2. Select the text on that Page you wish to turn into a link
  3. Click the control in the Link inspector
    Alternatively select Page > Create Link… (⌘L) from the menu bar
  4. Start typing to search for the Page you want
  5. Click on the Page you want to link to
    Alternatively use the up and down arrow keys to choose a Page hit the enter key to create a link to it
Image Page
  1. Start editing the Image Page you wish to create a link in
  2. Select or {[link|image_hotspots|create a hotspot]} on the image
  3. Click the control in the Link inspector
    Alternatively select Page > Create Link… (⌘L) from the menu bar
  4. Start typing to search for the Page you want
  5. Click on the Page you want to link to
    Alternatively use the up and down arrow keys to choose a Page hit the enter key to create a link to it
