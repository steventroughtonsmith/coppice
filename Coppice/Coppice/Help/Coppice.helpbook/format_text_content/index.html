Format Text Content

Text Pages support a wide range of formatting options. The Text inspector displays while editing a Text Page, giving access to these options.

Make text bold
  1. Start editing the Text Page
  2. Select some text
  3. Click the Bold {[image|format_text_content_bold.jpg|style="width:14px"]} button in the Text inspector
    Alternatively, select Format > Font > Bold (⌘B) from the menu bar
Italicise text
  1. Start editing the Text Page
  2. Select some text
  3. Click the Italic {[image|format_text_content_italic.jpg|style="width:14px"]} button in the Text inspector
    Alternatively, select Format > Font > Italic (⌘I) from the menu bar
Underline text
  1. Start editing the Text Page
  2. Select some text
  3. Click the Underline {[image|format_text_content_underline.jpg|style="width:14px"]} button in the Text inspector
    Alternatively, select Format > Font > Underline (⌘U) from the menu bar
Change the font
  1. Start editing the Text Page
  2. Select some text
  3. Click on the Font Family pop up button and choose a font family
  4. Click on the Typeface pop up button and choose the font within that family
    Alternatively, click the Aa button or select Format > Font > Show Font Panel… (⌘T) from the menu bar to get access to the full font panel
Change the text size
  1. Start editing the Text Page
  2. Select some text
  3. Click on the Text Size text field and enter a new size
    Alternatively, click on the stepper control to the right to increase or decrease the font size
    4. You can also increase or decrease font size by selecting Format > Font > Bigger (⌘+) or Format > Font > Smaller (⌘-) from the menu bar
Change text colour
  1. Start editing the Text Page
  2. Select some text
  3. Click on the colour well in the Text inspector
  4. Click on one of the colours in the popover
    Alternatively, click on the Show Color Picker… button to access the full macOS colour picker
Change text alignment
  1. Start editing the Text Page
  2. Select some text
  3. Click on a segment of the Text Alignment control
