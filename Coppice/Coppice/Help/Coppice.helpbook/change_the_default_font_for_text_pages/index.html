Change the Default Font for Text Pages

When creating a new Text Page, Coppice sets a default font. You can customise the family, typeface, and size of this font for all new Text Pages

Change the Default Font
  1. Select Coppice > Preferences… (⌘,) from the menu bar
  2. Click on the General tab
  3. Click the Change… button next to the Default Font
  4. Select a font family, typeface, and size from the Font Panel
Changing the default font does not affect any existing Text Pages.
