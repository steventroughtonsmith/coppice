Editing a Link

Text Page
  1. Start editing the Text Page you wish to edit a link in
  2. Move the text cursor to the link you want to edit
    3. You can option-click on a link to move the cursor there without activating the link
  3. Click the control in the Link inspector
    Alternatively select Page > Create Link… (⌘L) from the menu bar
  4. Start typing to search for the Page you now want to link to
  5. Click on the Page you want to link to
    Alternatively use the up and down arrow keys to choose a Page hit the enter key to create a link to it
Image Page
  1. Start editing the Image Page you wish to create a link in
  2. Select the Hotspot Mode icon {[image|image_hotspots_mode_icon.jpg|style="width:37px"]} from the image inspector
  3. Select the hotspot you wish the edit the link on
  4. Click the control in the Link inspector
    Alternatively select Page > Create Link… (⌘L) from the menu bar
  5. Start typing to search for the Page you want
  6. Click on the Page you want to link to
    Alternatively use the up and down arrow keys to choose a Page hit the enter key to create a link to it
