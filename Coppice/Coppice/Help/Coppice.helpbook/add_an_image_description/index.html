Add an Image Description

Image Pages in Coppice can have a description attached to an image. This can be used to add additional notes on the context of an image.

Change the image description
  1. Select an Image Page in the sidebar or on a Canvas
  2. Click on the Description field in the Image inspector on the right of the screen

The Image description will be used by Coppice to aid accessibility. If you plan to share a Coppice document with others, consider adding descriptions to any Image Pages
