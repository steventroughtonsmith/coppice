Customise the Sidebar Size

macOS allows you to change the size of sidebar items across the OS. By default Coppice uses this system setting. However, you can change the sidebar size just for Coppice

Change the Sidebar Size in Coppice
  1. Select Coppice > Preferences… (⌘,) from the menu bar
  2. Click on the General tab
  3. Click on the Sidebar size pop up button. Choose a size from the list ("Default" will use your system setting)
