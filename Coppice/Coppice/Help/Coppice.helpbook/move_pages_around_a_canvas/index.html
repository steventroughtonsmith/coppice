Move Pages Around a Canvas

Pages can be freely moved around a Canvas to help you better organise your thoughts

Move Pages using the mouse
  1. Click on a page (if the page is being edited you have to click on its title bar)
  2. Drag the page to where you wish to go
  3. Release the mouse button to finish moving
Moving the mouse cursor to the end of the Canvas editor while dragging will start scrolling the editor
If you have multiple pages selected, dragging one page will move all the selected pages
Move Pages using the keyboard
  1. {[link|select_pages|Select the Page(s)]} you wish to move
  2. Use the up, down, left, or right arrow key to move the Page(s) in that direct
Hold the shift key while hitting an arrow key to move the page further
