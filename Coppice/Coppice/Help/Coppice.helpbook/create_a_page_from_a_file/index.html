Create a Page From a File

Pages can be created from text or image files

Drag a file to Coppice
  1. Drag the file from Finder or another app
  2. Drop the file onto the sidebar on the left of the window
    Alternatively, drag onto a Canvas to both create a Page from that file and add it to the Canvas at the same time.
