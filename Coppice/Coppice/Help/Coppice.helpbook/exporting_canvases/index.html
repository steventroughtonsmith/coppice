Exporting Canvases

Coppice allows you to export a Canvas as a JPEG image, so you can share Canvases with anyone on any platform

Export canvas as an image
  1. Click on Canvases in the sidebar
  2. Click on the Canvas you wish to export
  3. Select File > Export Canvas as Imageâ€¦ from the menu bar
  4. Enter a name for the file and choose a location to save to
  5. Click Export
Exporting a Canvas as an image exports the Canvas as it is currently laid out. This means that any branches you have closed will not be visible. It also means that any content not currently visible on a page (due to that page being too small) will not be exported.
