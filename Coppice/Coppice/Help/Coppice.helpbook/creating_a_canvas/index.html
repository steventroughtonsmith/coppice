Creating a Canvas

Canvases are at the core of Coppice, allowing you to view and lay out multiple pages and see the relationships between them.

Create a new Canvas

Canvases can be created in one of 3 ways:

  • Click the New Canvas button in the toolbar
  • Select File > New Canvas (⌥⌘N) from the menu bar
  • Click the + button at the bottom of the Canvas List
The free version of Coppice only allows 1 Canvas per document. Upgrade to Coppice Pro to add multiple document to a Canvas
