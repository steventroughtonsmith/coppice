Re-order Pages

Pages can be ordered however you wish in the sidebar. This can help you keep track of all the Pages in your document.

Re-order Pages in the sidebar
  1. Find the Page you want to move in the sidebar
  2. Click and drag that Page. An indicator should appear as you drag the Page
  3. Release the mouse button when the indicator is where you want the Page to be
Automatically sort pages
  1. Right click on a Folder (or on the Pages header in the sidebar)
  2. Select Sort Folder by from the context menu
  3. Select how you want Coppice to sort the Pages
This will only sort Pages in the Folder you selected (or at the root level if you clicked on the Pages header). It won't sort any sub-Folders.
