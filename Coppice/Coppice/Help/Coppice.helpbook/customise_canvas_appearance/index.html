Customise Canvas Appearance

With a {[link|purchasing_coppice_pro|Coppice Pro subscription]}, you can customise various aspects of a Canvas's appearance

Change the theme
  1. Select Canvases from the sidebar
  2. Select the Canvas whos theme you want to change
  3. Click on the Theme pop up
  4. Choose the theme you wish to use for the Canvas
Toggle Page title visibility

By default Page titles only appear when you hover over a page. However, you can have them always visible.

  1. Select Canvases from the sidebar
  2. Select the Canvas you want to toggle the Page title visibility of
  3. Click on the Always Show Page Titles checkbox
