Removing a Link

Text Page
  1. Start editing the Text Page you wish to edit a link in
  2. Move the text cursor to the link you want to edit
    3. You can option-click on a link to move the cursor there without activating the link
  3. Click the {[image|create_a_link_remove_button.jpg|style="width:31px;" alt="Button containing a cross"]} button on the right of the Link inspector
Image Page
  1. Start editing the Image Page you wish to edit a link in
  2. Select the Hotspot Mode icon {[image|image_hotspots_mode_icon.jpg|style="width:37px"]} from the image inspector
  3. Select the hotspot you wish to remove the link from
  4. Click the {[image|create_a_link_remove_button.jpg|style="width:31px;" alt="Button containing a cross"]} button on the right of the Link inspector
    Alternative, hit the delete key to remove the hotspot as well as the link
