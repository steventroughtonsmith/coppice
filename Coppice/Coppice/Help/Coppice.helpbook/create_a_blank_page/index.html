Create a Blank Page

When creating a Page, you select the type of content that page will hold. Coppice supports two types of Pages: Text and Images. Coppice Documents can contain as many Pages as you like.

Creating a Page from the menu bar
  1. Select File > New Page in the menu bar
  2. Select the type of page to create
Creating a Page from the toolbar
  1. Click and hold on the New Page {[image|create_a_blank_page_toolbar_icon.jpg|style="width:18px"]} toolbar item
  2. Select the type of page to create
The toolbar remembers the last type of page you created. You can quickly create another page of the same type by simply clicking on the New Page toolbar item
Creating a Page from the Page Selector
  1. Open a Page Selector (either by {[link|view_a_page|jumping to a page]} or {[link|add_a_page_to_a_canvas|adding a page to a canvas]})
  2. Type the title you want for the new Page

  4. Hit the Enter key to create the new page
Performing these actions while viewing a Canvas will also add the new Page to that Canvas
