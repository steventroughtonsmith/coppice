Tables

Text Pages in Coppice support adding tables. Tables give you more control over formatting structured content

Creating a table
  1. Start editing a Text Page
  2. Select Format > Table… from the menu bar
Changing table size
  1. Start editing a Text Page
  2. Move the text cursor to the table you wish to change
  3. Select Format > Table… from the menu bar to bring up the table panel
  4. Change the Row and Column fields to resize the table
    5. You can also merge and split cells using the buttons on the right
Changing cell alignment
  1. Start editing a Text Page
  2. Move the text cursor to the cell you wish to change the alignment of
  3. Select Format > Table… from the menu bar to bring up the table panel
  4. Change the horizontal alignment using the segmented control on the left of the Alignment section
  5. Change the vertical alignment using the segmented control on the right of the Alignment section
Changing cell border
  1. Start editing a Text Page
  2. Move the text cursor to the cell you wish to change the border of
  3. Select Format > Table… from the menu bar to bring up the table panel
  4. Change the border width by editing the text field in the Cell Border section
  5. Change the border colour by using the colour well in the Cell Border section
Changing cell background
  1. Start editing a Text Page
  2. Move the text cursor to the cell you wish to change the background of
  3. Select Format > Table… from the menu bar to bring up the table panel
  4. Select the type of background using the pop up button in the Cell Background section
  5. Change the background colour using the colour well in the Cell Border section
