Add an External Link

Pages can contain links to external locations such as websites

Text Page
  1. Start editing the Text Page you wish to create a link in
  2. Select the text on that Page you wish to turn into a link
  3. Click on the control in the Link inspector
  4. Enter the URL of the link you wish to add
  5. Select External Link from the Create New… section of auto complete and hit Enter
Image Page
  1. Start editing the Image Page you wish to create a link in
  2. Select or {[link|image_hotspots|create a hotspot]} on the image
  3. Click on the control in the Link inspector
  4. Enter the URL of the link you wish to add
  5. Select External Link from the Create New… section of auto complete and hit Enter
