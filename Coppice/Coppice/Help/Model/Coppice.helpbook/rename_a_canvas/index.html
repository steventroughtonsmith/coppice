Rename a Canvas

Each Canvas can have a title, to help you more easily distinguish Canvases in the Canvas List.

Edit a Canvas's title

There are multiple ways of changing a Canvas's title

  • In the Canvas List:
    1. Select Canvases in the sidebar
    2. Double-click on the Canvas's title in the Canvas List
    3. Type the new title
    4. Hit enter
  • In the Inspector:
    1. Select Canvases in the sidebar
    2. Select the Canvas who's title you want to edit in the Canvas List
    3. Click on the Title field in the Canvas inspector
    4. Type the new title
    5. Hit enter or navigate away from the Title field
