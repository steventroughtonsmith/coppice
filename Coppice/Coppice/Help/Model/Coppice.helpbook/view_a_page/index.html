View a Page

Pages can be displayed in the main editor by selecting them from the sidebar, allowing you to focus on one Page at a time.

Jump to a Page
  1. Select Page > Jump to Page… (⇧⌘J) from the menu bar
  2. Enter search text to filter pages down
  3. Use the up and down arrow keys to select the page and hit the Enter key to open that page
    Alternatively, click on a page with the mouse
