Add Text to a Page

Whether blank or created from an existing file, you can edit Text Pages from directly within Coppice.

Edit a Text Page in a focused editor
  1. {[link|view_a_page|Select the Page]} in the sidebar
  2. Click on the text area in the editor
  3. Start typing
Edit a Text Page on a Canvas
  1. Find the Page on the Canvas
  2. Double-click on the text of the page where you want to start editing
  3. Start typing
